# nclicker-rs

nclicker - Windows Auto-Clicker in Rust
A modern Windows auto-clicker application built with Rust, featuring a clean GUI.

Features
랜 Precise Click Intervals - Configure timing down to milliseconds
랜 Mouse Button Selection - Left or Right mouse button
랜 Click Types - Single or Double clicks
랜 Flexible Repeat Options - Fixed count or repeat until stopped
랜 Cursor Positioning - Current location or specific coordinates
랜 Random Offset - Add timing variation to avoid detection
랜 Light/Dark Themes - Switch between UI themes
랜 Real-time Status - Shows running state and click count
랜 Thread-safe Design - Non-blocking UI during operation
